# CVerify 
## How to Run the Project

1. Clone the repository:

   ```bash
   git clone <repo-url>
   ```

2. Navigate to the project folder:

   ```bash
   cd frontend/cverify
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Start the development server:

   ```bash
   npm run dev
   ```

5. Open in browser:

   ```
   http://localhost:5173
   ```
